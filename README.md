# 简易通讯录后台管理系统 java754

## 项目概述

简易通讯录后台管理系统是一个基于SSM框架（Spring+SpringMVC+Mybatis）的开发项目，适用于学习或项目实战。该项目提供全面的通讯录管理功能，包括数据的增删改查，并具备模糊搜索能力。

## 技术栈

- **后端框架**：Spring、SpringMVC、Mybatis
- **项目构建工具**：Maven
- **服务器**：Tomcat
- **数据库**：MySQL
- **数据库连接池**：C3P0
- **日志管理**：log4j

## 功能模块

- 数据增加
- 数据删除
- 数据修改
- 数据查询
- 模糊搜索

## 系统角色

本项目主要面向具有后台管理系统需求的用户。

## 运行环境

- **操作系统**：兼容Windows、Linux等常见操作系统
- **JDK版本**：1.8或以上
- **数据库**：MySQL 5.5或以上
- **服务器**：Tomcat 8.0或以上

## 部署步骤

1. 环境准备：确保JDK、MySQL、Tomcat已正确安装并配置环境变量。
2. 数据库构建：执行项目中提供的SQL脚本，以创建数据库和表结构。
3. 项目编译：使用Maven对项目进行编译，确保无错误。
4. 部署到Tomcat：将编译后的项目部署到Tomcat服务器。
5. 访问系统：通过浏览器访问系统，默认端口为8080。
